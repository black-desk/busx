# busx Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `busx`, a single-binary zbus-based D-Bus CLI (dbus-send/busctl/qdbus replacement) that lists/introspects the bus, calls methods, gets/sets properties, and monitors signals — emitting type-tagged JSON / NDJSON and shipping dynamic shell completion.

**Architecture:** One Rust binary crate (no published lib), `src/main.rs` + focused modules under `src/`. Uses zbus's blocking API. Input values use busctl-style signature+positional text; output is always type-tagged JSON (`{"type":..,"data":..}`), monitor is NDJSON. Default bus tries session then falls back to system. Testing is end-to-end only: each integration test spins up one private `dbus-daemon` + a zbus test service (shared via a `OnceLock` fixture) and drives the `busx` binary with `assert_cmd`.

**Tech Stack:** Rust (edition 2024), `zbus 5` (blocking), `zvariant`, `clap 4` (derive) + `clap_complete`, `serde`/`serde_json`, `roxmltree` (introspection XML), `thiserror`/`anyhow`; dev: `assert_cmd`, `predicates`; system dep: `dbus-daemon`.

**Spec:** `docs/superpowers/specs/2026-07-07-busx-design.md`

> **API verification note (read before every code task):** zbus 5.16 / zvariant signatures below follow documented 5.x patterns but may not be byte-perfect. After writing each task's code, run `cargo build` / `cargo test`; if a zbus/zvariant method name or signature mismatches, look it up on <https://docs.rs/zbus/5.16.0> / <https://docs.rs/zvariant> and adjust. The TDD loop (red→green) is what guarantees correctness, not the snippets here.

## File Structure

- `Cargo.toml` — manifest, deps (§13 of spec).
- `src/main.rs` — entry: clap parse → dispatch to `ops::*`; owns global flags (`--user/--system/--address/--verbose`).
- `src/cli.rs` — clap `Cli`/`Command` derive definitions for all subcommands.
- `src/conn.rs` — `connect()` with session→system fallback; respects explicit bus flags.
- `src/error.rs` — `busx::Error` (`thiserror`); `exit_code()` → 0/1; print as `busx: <msg>` to stderr.
- `src/out.rs` — emit type-tagged JSON / NDJSON to stdout.
- `src/value/` —
  - `encode.rs` — busctl text → `zvariant::Value`.
  - `decode.rs` — `zvariant::Value` → type-tagged `serde_json::Value`.
  - `dict.rs` — non-string-key dict safe rendering (array-of-pairs).
- `src/introspect.rs` — parse introspection XML (`roxmltree`) → structured types.
- `src/ops/` — one file per subcommand: `list.rs`, `introspect.rs`, `tree.rs`, `call.rs`, `property.rs` (get/set), `monitor.rs`.
- `src/complete.rs` — dynamic completion candidate generation + hidden `__complete` subcommand.
- `tests/common/mod.rs` — shared `dbus-daemon` + zbus test-service fixture.
- `tests/<command>.rs` — one e2e test file per subcommand.
- `.github/workflows/ci.yaml` — uncomment the `rust` job.
- `README.md` / `README.zh_CN.md` — real busx README (template updated, roadmap written in).

Decomposition rule: each task below is independently testable end-to-end and ends with a commit.

```rust
use std::process::{Command, Stdio};
use std::sync::OnceLock;
use zbus::blocking::Connection;
use zbus::interface;

/// Test service interface. Methods/properties are chosen to exercise every
/// busx feature: nested encode (TakeHints a{sv}), decode of basic + non-string-key
/// dict (Counts a{uu}), settable property that emits PropertiesChanged (Volume).
pub struct TestIface {
    volume: f64,
}

#[interface(name = "org.busx.Test")]
impl TestIface {
    #[zbus(property)]
    fn volume(&self) -> f64 {
        self.volume
    }
    #[zbus(property)]
    fn set_volume(&mut self, v: f64) {
        self.volume = v;
    }

    #[zbus(property)]
    fn name(&self) -> String {
        "busx-test".into()
    }

    /// Non-string-key dict property: a{uu}. Exercises the §7.2 rule.
    #[zbus(property)]
    fn counts(&self) -> std::collections::HashMap<u32, u32> {
        [(1u32, 10u32), (2, 20)].into_iter().collect()
    }

    /// String-key dict-of-variant property: a{sv}.
    #[zbus(property)]
    fn hints(&self) -> std::collections::HashMap<String, zvariant::Value> {
        [("urgency".to_string(), zvariant::Value::new(1u8))].into_iter().collect()
    }

    /// Method that takes a{sv} and returns the entry count — exercises encode.
    fn take_hints(&self, hints: std::collections::HashMap<String, zvariant::Value>) -> u32 {
        hints.len() as u32
    }

    /// Echo method for nested array encode test.
    fn join(&self, parts: Vec<String>) -> String {
        parts.join("-")
    }

    /// Bump volume to trigger PropertiesChanged for monitor tests.
    fn bump_volume(&mut self) -> f64 {
        self.volume += 1.0;
        self.volume
    }
}

pub struct TestBus {
    pub address: String,
    _conn: Connection,
    daemon_pid: u32,
}

impl Drop for TestBus {
    fn drop(&mut self) {
        let _ = nix::sys::signal::kill(
            nix::unistd::Pid::from_raw(self.daemon_pid as i32),
            nix::sys::signal::Signal::SIGTERM,
        );
    }
}

static BUS: OnceLock<TestBus> = OnceLock::new();

pub fn bus() -> &'static TestBus {
    BUS.get_or_init(|| {
        // --fork: parent prints address + pid to stdout then exits; daemon detaches.
        let out = Command::new("dbus-daemon")
            .args(["--session", "--nofork", "--print-address=1", "--print-pid=1"])
            .stdout(Stdio::piped())
            .stderr(Stdio::null())
            .spawn()
            .expect("dbus-daemon not found; apt install dbus");
        // We used --nofork, so the child stays foreground; read address+pid lines.
        let mut child = out;
        use std::io::{BufRead, BufReader};
        let mut reader = BufReader::new(child.stdout.take().expect("piped"));
        let mut address = String::new();
        reader.read_line(&mut address).expect("read address");
        let address = address.trim().to_string();
        let mut pid = String::new();
        reader.read_line(&mut pid).expect("read pid");
        let daemon_pid: u32 = pid.trim().parse().expect("pid number");

        let conn = Connection::address(&address).expect("connect test bus");
        conn.object_server()
            .at("/org/busx/Test", TestIface { volume: 0.5 })
            .expect("register object");
        conn.request_name("org.busx.Test").expect("request name");

        TestBus { address, _conn: conn, daemon_pid }
    })
}
```

```rust
use crate::error::{Error, Result};
use zbus::blocking::Connection;

/// Connect per the explicit flag, else try session then fall back to system.
pub fn connect(user: bool, system: bool, address: Option<&str>, verbose: bool) -> Result<Connection> {
    if let Some(addr) = address {
        return Ok(Connection::address(addr)?);
    }
    if system {
        return Ok(Connection::system()?);
    }
    // user or default: try session first.
    match Connection::session() {
        Ok(c) => Ok(c),
        Err(e) if user => Err(Error::Msg(format!("cannot connect to session bus: {e}"))),
        Err(e) => {
            if verbose {
                eprintln!("busx: warning: session bus unavailable ({e}); falling back to system bus");
            }
            Ok(Connection::system()?)
        }
    }
}
```

```rust
use serde_json::{json, Value as Json};
use zvariant::Value;

/// Render a zvariant::Value as type-tagged JSON: {"type": <sig>, "data": <native>}.
pub fn to_tagged(v: &Value) -> Json {
    match v {
        Value::U8(b) => tagged("y", json!(b)),
        Value::Bool(b) => tagged("b", json!(b)),
        Value::I16(i) => tagged("n", json!(i)),
        Value::U16(i) => tagged("q", json!(i)),
        Value::I32(i) => tagged("i", json!(i)),
        Value::U32(i) => tagged("u", json!(i)),
        Value::I64(i) => tagged("x", json!(i)),
        Value::U64(i) => tagged("t", json!(i)),
        Value::F64(d) => tagged("d", json!(d)),
        Value::Str(s) => tagged("s", json!(s.as_str())),
        Value::Signature(s) => tagged("g", json!(s.to_string())),
        Value::ObjectPath(o) => tagged("o", json!(o.as_str())),
        Value::Value(inner) => tagged("v", to_tagged(inner)),
        Value::Array(a) => {
            let elem = a.element_signature().to_string();
            let sig = format!("a{elem}");
            let data = Json::Array(a.iter().map(to_tagged).collect());
            tagged(&sig, data)
        }
        Value::Structure(s) => {
            // build signature like "(...)"
            let sig = format!("({})", s.fields().map(|f| field_sig(f)).collect::<String>());
            let data = Json::Array(s.fields().map(to_tagged).collect());
            tagged(&sig, data)
        }
        Value::Dict(d) => crate::value::dict::to_tagged(d),
        other => tagged("?", json!(format!("unsupported: {other:?}"))), // fd (h), etc.
    }
}

fn tagged(ty: &str, data: Json) -> Json {
    json!({ "type": ty, "data": data })
}

/// Best-effort single-complete-type signature string for a value (used for struct fields).
fn field_sig(v: &Value) -> String {
    match v {
        Value::U8(_) => "y".into(), Value::Bool(_) => "b".into(),
        Value::I16(_) => "n".into(), Value::U16(_) => "q".into(),
        Value::I32(_) => "i".into(), Value::U32(_) => "u".into(),
        Value::I64(_) => "x".into(), Value::U64(_) => "t".into(),
        Value::F64(_) => "d".into(), Value::Str(_) => "s".into(),
        Value::Signature(_) => "g".into(), Value::ObjectPath(_) => "o".into(),
        Value::Array(a) => format!("a{}", a.element_signature()),
        Value::Structure(s) => format!("({})", s.fields().map(field_sig).collect::<String>()),
        Value::Dict(d) => format!("a{{{}{}}}", d.key_signature(), d.value_signature_or(None).unwrap_or_default().as_str()),
        Value::Value(inner) => format!("v"),
        _ => "?".into(),
    }
}
```

`src/value/dict.rs` — the safety rule:
```rust
use serde_json::{json, Value as Json};
use zvariant::Dict;

/// String-key dict -> JSON object; non-string-key dict -> array of {key,value} pairs.
/// Never panics on non-string keys (unlike sd-bus #32904).
pub fn to_tagged(d: &Dict) -> Json {
    let key_sig = d.key_signature();
    let val_sig = d.value_signature_or(None).map(|s| s.to_string()).unwrap_or_default();
    let sig = format!("a{{{key_sig}{val_sig}}}");
    if key_sig == "s" || key_sig == "o" || key_sig == "g" {
        // string-like keys: JSON object
        let mut obj = serde_json::Map::new();
        for (k, v) in d.iter() {
            let key = match k {
                zvariant::Value::Str(s) => s.to_string(),
                zvariant::Value::ObjectPath(o) => o.to_string(),
                zvariant::Value::Signature(s) => s.to_string(),
                _ => k.to_string(),
            };
            obj.insert(key, crate::value::decode::to_tagged(v));
        }
        json!({ "type": sig, "data": Json::Object(obj) })
    } else {
        // non-string keys: array of pairs (key kept as native JSON type)
        let data = d.iter()
            .map(|(k, v)| json!({"key": crate::value::decode::to_tagged(k), "value": crate::value::decode::to_tagged(v)}))
            .collect::<Vec<_>>();
        json!({ "type": sig, "data": data })
    }
}
```

```rust
pub fn get(user: bool, system: bool, address: Option<&str>, verbose: bool,
           service: &str, object: &str, interface: Option<&str>, props: &[String]) -> Result<()> {
    let conn = connect(user, system, address, verbose)?;
    let proxy = PropertiesProxy::new(&conn, service, object)?;
    let iface = interface.unwrap_or("");
    if props.is_empty() {
        // GetAll
        let map = proxy.get_all(iface)?;
        let mut obj = serde_json::Map::new();
```

Recursive descent over the signature string, consuming tokens from a `slice` cursor. Mirrors busctl: arrays need a count, variants need an inner signature, structs/dict-entries flatten.

```rust
use crate::error::{Error, Result};
use zvariant::{Value, Signature, ObjectPath, Array, Dict, Structure};

struct Cursor<'a> { tokens: &'a [String], pos: usize }

impl<'a> Cursor<'a> {
    fn next(&mut self) -> Result<&str> {
        let t = self.tokens.get(self.pos).ok_or_else(|| Error::Msg("not enough arguments".into()))?;
        self.pos += 1;
        Ok(t.as_str())
    }
    fn done(&self) -> bool { self.pos >= self.tokens.len() }
}

/// Parse a full signature from the trailing tokens. First token is the signature
/// string, then values per busctl positional rules.
pub fn parse(tokens: &[String]) -> Result<Vec<Value>> {
    if tokens.is_empty() { return Ok(vec![]); }
    let mut sig = tokens[0].chars().peekable();
    let mut cur = Cursor { tokens: &tokens[1..], pos: 0 };
    let mut out = Vec::new();
    while let Some(c) = sig.next() {
        out.push(parse_one(c, &mut sig, &mut cur)?);
    }
    if !cur.done() {
        return Err(Error::Msg(format!("{} extra argument(s) given", cur.tokens.len() - cur.pos)));
    }
    Ok(out)
}

fn parse_one(c: char, sig: &mut std::iter::Peekable<std::str::Chars>, cur: &mut Cursor) -> Result<Value> {
    let v = match c {
        'y' => Value::U8(cur.next()?.parse().map_err(|e| Error::Msg(format!("byte: {e}")))?),
        'b' => Value::Bool(parse_bool(cur.next()?)),
        'n' => Value::I16(cur.next()?.parse().map_err(|e| Error::Msg(format!("int16: {e}")))?),
        'q' => Value::U16(cur.next()?.parse().map_err(|e| Error::Msg(format!("uint16: {e}")))?),
        'i' => Value::I32(cur.next()?.parse().map_err(|e| Error::Msg(format!("int32: {e}")))?),
        'u' => Value::U32(cur.next()?.parse().map_err(|e| Error::Msg(format!("uint32: {e}")))?),
        'x' => Value::I64(cur.next()?.parse().map_err(|e| Error::Msg(format!("int64: {e}")))?),
        't' => Value::U64(cur.next()?.parse().map_err(|e| Error::Msg(format!("uint64: {e}")))?),
        'd' => Value::F64(cur.next()?.parse().map_err(|e| Error::Msg(format!("double: {e}")))?),
        's' => Value::Str(cur.next()?.into()),
        'o' => Value::ObjectPath(ObjectPath::try_from(cur.next()?)?),
        'g' => Value::Signature(Signature::try_from(cur.next()?)?),
        'v' => {
            // variant: next token is the inner signature, then its value
            let inner_sig = cur.next()?;
            let mut inner = inner_sig.chars().peekable();
            let one = parse_one(inner.next().ok_or_else(|| Error::Msg("empty variant sig".into()))?, &mut inner, cur)?;
            Value::Value(Box::new(one))
        }
        'a' => {
            // array: next token is the element count, then that many elements
            let n: usize = cur.next()?.parse().map_err(|e| Error::Msg(format!("array count: {e}")))?;
            // peek element signature: it may be multi-char ({kv} or (...))
            let elem_sig_str = read_one_complete_type(sig)?;
            let mut elem_chars = elem_sig_str.chars().peekable();
            let mut arr = Vec::new();
            for _ in 0..n {
                let mut ec = elem_chars.clone();
                arr.push(parse_one(ec.next().unwrap(), &mut ec, cur)?);
            }
            Value::Array(Array::from_iter(arr))
        }
        '(' => {
            // struct: read until ')'
            let mut fields = Vec::new();
            loop {
                match sig.peek() {
                    Some(')') => { sig.next(); break; }
                    Some(_) => {
                        let f = parse_one(sig.next().unwrap(), sig, cur)?;
                        fields.push(f);
                    }
                    None => return Err(Error::Msg("unterminated struct".into())),
                }
            }
            Value::Structure(Structure::try_from(fields)?)
        }
        '{' => Err(Error::Msg("dict-entry outside array".into())),
        other => return Err(Error::Msg(format!("unsupported type code '{other}'"))),
    };
    Ok(v)
}

fn parse_bool(s: &str) -> bool {
    matches!(s, "true" | "yes" | "on" | "1")
}

/// Read one complete D-Bus type from the signature cursor, handling 'a{kv}' and 'a(...)'.
fn read_one_complete_type(sig: &mut std::iter::Peekable<std::str::Chars>) -> Result<String> {
    match sig.peek() {
        Some('{') => {
            sig.next();
            let mut depth = 1;
            let mut s = String::from("{");
            while depth > 0 {
                match sig.next() { Some('}') => { depth -= 1; s.push('}'); } Some(c) => s.push(c), None => break }
            }
            Ok(s)
        }
        Some('(') => {
            sig.next();
            let mut depth = 1;
            let mut s = String::from("(");
            while depth > 0 {
                match sig.next() { Some(')') => { depth -= 1; s.push(')'); } Some(c) => s.push(c), None => break }
            }
            Ok(s)
        }
        Some(c) => { sig.next(); Ok(c.to_string()) }
        None => Err(Error::Msg("expected element type".into())),
    }
}
```

The dict case (`a{kv}`): when the element signature is `{kv}`, the array loop reads `n` pairs; each pair is key (one complete type) then value (one complete type). The `parse_one` for the `{` element needs handling — extend the `'a'` arm: when `elem_sig_str` starts with `{`, for each of the `n` entries, parse the key type then the value type.
```rust
use serde_json::{json, Value as Json};

pub fn parse_xml(xml: &str) -> Json {
    let doc = match roxmltree::Document::parse(xml) { Ok(d) => d, Err(e) => return json!({"error": e.to_string()}) };
    let mut ifaces = Vec::new();
    for iface in doc.descendants().filter(|n| n.has_tag_name("interface")) {
        let mut methods = Vec::new();
        let mut signals = Vec::new();
        let mut props = Vec::new();
        for m in iface.children().filter(|n| n.has_tag_name("method")) {
            methods.push(json!({"name": m.attribute("name"), "in": members(&m, "in"), "out": members(&m, "out")}));
        }
        for s in iface.children().filter(|n| n.has_tag_name("signal")) {
            signals.push(json!({"name": s.attribute("name"), "args": members(&s, "arg")}));
        }
        for p in iface.children().filter(|n| n.has_tag_name("property")) {
            props.push(json!({"name": p.attribute("name"), "type": p.attribute("type"), "access": p.attribute("access")}));
        }
        ifaces.push(json!({
            "name": iface.attribute("name"),
            "methods": methods, "signals": signals, "properties": props,
        }));
    }
    json!(ifaces)
}

fn members(method: &roxmltree::Node, dir: &str) -> Vec<Json> {
    method.children().filter(|n| n.has_tag_name("arg"))
        .filter(|a| a.attribute("direction").map(|d| d == dir).unwrap_or(dir == "arg"))
        .map(|a| json!({"name": a.attribute("name"), "type": a.attribute("type")}))
        .collect()
}
```

```rust
use crate::conn::connect;
```

```rust
fn walk(conn: &Connection, service: &str, path: &str, out: &mut Vec<String>) -> Result<()> {
    out.push(path.to_string());
    let xml = IntrospectableProxy::new(conn, service, path)?.introspect()?;
    let doc = roxmltree::Document::parse(&xml)?;
    for child in doc.descendants().filter(|n| n.has_tag_name("node")) {
        if let Some(name) = child.attribute("name") {
            if name.starts_with('/') { continue; } // absolute — skip
            let child_path = format!("{}/{}", path.trim_end_matches('/'), name);
            walk(conn, service, &child_path, out)?;
        }
    }
    let _ = Value::U8(0);
    Ok(())
}

pub fn run(user: bool, system: bool, address: Option<&str>, verbose: bool, services: &[String]) -> Result<()> {
    let conn = connect(user, system, address, verbose)?;

        zbus::blocking::fdo::DBusProxy::new(&conn)?.list_names()?
            .into_iter().filter(|n| !n.starts_with(':')).map(Into::into).collect()
    } else { services.to_vec() };
    let mut tree = serde_json::Map::new();
    for svc in services {
        let mut paths = Vec::new();
        let _ = walk(&conn, &svc, "/", &mut paths);
        tree.insert(svc, json!(paths));
    }
    crate::out::print_json(&json!(tree));
    Ok(())
}
```
